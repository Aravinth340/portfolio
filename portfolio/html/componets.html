COMPONENTS OF COMPUTER

MONITOR

A computer monitor is an output device that displays information in pictorial or textual form. A discrete monitor comprises a visual display, support electronics, power supply, housing, electrical connectors, and external user controls.

CPU

CPU stands for central processing unit. It is the soul of a computer system. The CPU along with the memory and the I/O sub-system establish a strong computer system.

KEYBOARD

A computer keyboard is an input device used to enter characters and functions into the computer system by pressing buttons, or keys. It is the primary device used to enter text.

MOUSE

A computer mouse is a hand-held pointing device that detects two-dimensional motion relative to a surface. This motion is typically translated into the motion of the pointer (called a cursor) on a display, which allows a smooth control of the graphical user interface of a computer.
